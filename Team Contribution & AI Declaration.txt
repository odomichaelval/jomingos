# Team Effort Contribution

This project was collaboratively developed by Joshua Onwuka and Michael Odo.

Both team members jointly contributed to the planning, development, testing, deployment, and presentation of the project. Contributions included:

Backend system development and API implementation,
Frontend design and user interface development,
Database integration and system configuration,
Deployment and hosting configuration,
Preparation of presentation slides,
Creation of the project explanation/demo video,
Technical documentation and report preparation,
System testing, debugging, and refinement.

The project was completed through shared collaboration, active communication, and combined technical efforts from both contributors throughout the development lifecycle.



# AI Declaration


## AI for Developing

Artificial Intelligence (AI) tools were used during the development of this project to support the enhancement of concepts, implementation approaches, technical problem-solving, and content refinement.

AI was utilised to assist with:

* Generating and refining development ideas,
* Improving coding structure and debugging processes,
* Enhancing documentation and written explanations,
* Refining user interface concepts,
* Supporting the development of project outputs.

The use of AI contributed to the detailed development of several aspects of the project; however, all final decisions, implementation choices, testing, evaluation, and critical reviews were conducted by us.

We maintained significant involvement throughout the project by reviewing, modifying, validating, and refining AI-generated suggestions before inclusion in the final system and documentation. AI-generated content was critically assessed and adapted where necessary to ensure accuracy, relevance, originality, and alignment with the project objectives.

The final outputs therefore represent a combination of AI-assisted development and substantial human input, judgement, refinement, and evaluation.
